Before You Install Microbot Launcher

Why these steps?
The app is open source and not signed with Apple (paid) certificates. macOS adds a "quarantine" flag to downloaded apps. Removing that flag stops the "damaged / cannot be opened" warning.
  1. On the Microbot Launcher Installer window, drag Microbot Launcher.app on Applications folder shortcut.
  2. Open Terminal:
    Press Command (⌘) + Space, type Terminal, press Enter.
  3. Run this command exactly (copy & paste): 
    xattr -cr "/Applications/Microbot Launcher.app"
    (It silently removes the quarantine flag. No output means success.)
  4. Launch the app: Open Launchpad or Finder → Applications → Microbot Launcher.
Alternative (one-time): You can also right-click the app → Open → Open. If you still get the damaged message, run the command above.

You only need to do this once per version. Future updates you download manually may require repeating the command.
